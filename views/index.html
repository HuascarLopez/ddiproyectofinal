Proyecto Final DDI

Curso 2020-1

Introducción

La presente live app es donde se presentará el proyecto final para la evaluación a titulo de suficiencia del curso 'Diseño de interfaces' en CETYS Universiad.

Resumen
Se creará un sistema de monitoreo de niveles de luz en habitaciones de una casa utilizando un foto resistor, placas de desarrollo, módulos WIFI y un desarrollo web con Node.js and Express.js. Aplicando el protocolo de comunicación WIFI y la implementación de una REST API para cumplir con el diseño de una interfaz.

Objetivos
Objetivo general
Establecer un sistema de monitoreo de los niveles de luz de una casa. Objetivos específicos
Recopilar señales digitales empleando sensores y placas de desarrollo. Utilizar un protocolo de comunicación serial. Registrar los datos a partir de esta comunicación. Generar un tablero de información.

Planteamiento del problema
Existiendo una casa con tres habitaciones principales compartidas: sala, cocina y baño se busca registrar la cantidad de tiempo en que los focos estuvieron encendidos. Procedimiento:

  • Obtener señal de foto resistor a placa de desarrollo.
  • Enviar (POST) por módulo WIFI la información a un servidor web.
  • Convertir datos a un formato JSON (id, cuando se envió, nivel de la señal obtenida, una codificación para definir si esta encendido o apagado) y generar REST API.
  • Generar una aplicación en una plataforma de construcción y escalamiento de aplicaciones donde se pueda ver un dashboard de esta información.

Entregables:

  • Presentación en PowerPoint de la información general.
  • Reporte en .pdf de la información detallada.
  • Documentación del código utilizado y repositorio de la aplicación.
